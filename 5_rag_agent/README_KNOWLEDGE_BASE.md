# 知识库文档说明

本目录用于存放知识库文档，支持以下格式：

- `.txt` - 纯文本文件
- `.md` - Markdown 文件
- `.pdf` - PDF 文档
- `.docx` - Word 文档

## 建议的文档组织方式

```
knowledge_base/
├── python/
│   ├── basics.md
│   ├── functions.md
│   └── oop.md
├── database/
│   ├── mysql.md
│   ├── postgresql.md
│   └── redis.md
└── tools/
    ├── git.md
    └── docker.md
```

## 文档要求

1. 每个文档应聚焦一个主题
2. 文档大小建议：1-5 KB
3. 使用清晰的标题层级
4. 避免过于冗长的段落

## 示例文档

可以创建以下示例文档进行测试：

1. `python_basics.md` - Python 基础语法
2. `database_config.md` - 数据库配置说明
3. `git_usage.md` - Git 使用指南
4. `docker_intro.md` - Docker 入门教程
